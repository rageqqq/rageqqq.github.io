All-in-OneNUS

I worked on a CLI Application that aids in university life using Java.

Using Java, I built a CLI Application feature that helps the user gather food recommendations. This feature allows a user to get food recommendations from different locations within NUS, and modify the list if needed. It also allows the user to add dietary restrictions such as halal to each food place.
