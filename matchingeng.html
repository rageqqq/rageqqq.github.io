Stock Order Matching

I worked on a Stock Order Matching Engine using C++ which allows for two way communication using TCP/IP.

Using C++, I built a Stock Order Matching Application that utilizes a TCP/IP Connection for communication between Client and Server. The Client sends Buy/Sell Stock Orders to the Server which will respond with any matches. The Application comes completes with basic testing and a high speed stock order generator to allow for simulation of the Matching Engine.
